#Cat or Dog CNN

This notebook and dataset trains the model to determine the difference between images of cats and of dogs.

Image transformation is used to increase the accuracy of the model without over training it on the training set.

##To run this project

You must have Jupyter Notebooks configured on your computer,
The data set and proccessing power required are too high for google colaboratory to run it.
The file paths must be changed to reflect where you store the notebook and data set folders.
